## 1️⃣ Project Name & Problem Statement
**(Speaker 1: Introduction)**

"Assalam-o-Alaikum everyone! Humari team ka naam hai [Team Name] aur aaj hum introduce kar rahe hain **MehfoozAI**.

**Problem Statement:** Pakistan mein harassment reporting ek bada challenge hai. Social stigma, retaliation ka dar, aur legal complexity ki wajah se hazaron incidents report hi nahi hote. Women feel unsafe, aur unke paas koi aisa platform nahi hai jahan wo bina kisi dar ke, anonymously apni baat keh sakein."

---

## 2️⃣ Problem Solution
**(Speaker 2: The Solution)**

"Is masle ka solution hai **MehfoozAI**. Humne ek AI-powered anonymous reporting platform banaya hai jo directly WhatsApp par kaam karta hai.

User ko sirf ek message ya voice note bhejna hota hai. Humara AI engine us raw description ko analyze karta hai, Pakistan Penal Code (PPC) ke relevant sections map karta hai, aur ek legal FIR draft ready kar deta hai. Isse reporting ka process anonymous bhi rehta hai aur legally structured bhi."

---

## 3️⃣ Technology Stack Used
**(Speaker 3: Technical Details)**

"Humne is project ko build karne ke liye modern aur scalable tech stack use kiya hai:
- **Frontend:** Next.js 14 (App Router) with Vanilla CSS for a premium glassmorphic UI.
- **Backend:** FastAPI (Python) for high-speed asynchronous processing.
- **Database:** Supabase (PostgreSQL) with Realtime capabilities.
- **AI Engine:** LangGraph aur custom pipelines (OpenAI/Gemini) intelligence extraction ke liye.
- **Voice:** Whisper AI for voice-to-text transcription.
- **Messaging:** Meta WhatsApp Cloud API for the primary interface."

---

## 4️⃣ High-Level Diagram
**(Speaker 4: System Architecture)**

"System ka workflow kaafi simple aur robust hai:
1. User **WhatsApp** par report bhejta hai.
2. Humara **FastAPI Backend** use receive karta hai aur AI pipeline ko trigger karta hai.
3. AI data extract karke **Supabase Database** mein save karta hai.
4. Authorities ke liye ek **Next.js Dashboard** hai jahan wo real-time incidents aur safety heatmaps dekh sakte hain."
