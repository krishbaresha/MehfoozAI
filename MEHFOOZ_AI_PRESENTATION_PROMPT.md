## 🚀 Project Overview
- **Name:** MehfoozAI (محفوظ اے آئی)
- **Tagline:** Pakistan's First Anonymous Harassment Reporting AI.
- **Problem we are solving:** Pakistan mein harassment ya emergency ke waqt log police station jane se darte hain (societal pressure, identity reveal hone ka khauf, complex legal system).
- **Our Solution:** Ek WhatsApp bot jahan victim anonymous reh kar message ya voice note bhej sakti hai. AI automatically waqia samajh kar FIR draft karta hai, PPC (Pakistan Penal Code) sections lagata hai, aur relevant authorities (Police/NGO) ko route karta hai.

## 🛠️ Key Features (Core Tech)
1. **WhatsApp Intake:** Koi nayi app download karne ki zaroorat nahi. Har kisi ke paas WhatsApp hai. Victim text, voice note, ya live location bhej sakti hai.
2. **100% Anonymity:** Identity reveal nahi hoti. Number mask ho jata hai.
3. **AI Intelligence Engine (Groq + Llama 3):** 
   - Voice notes ko text mein transcribe karta hai (Whisper).
   - Incident se facts extract karta hai.
   - Automatically legal PPC sections (e.g., Section 509, 354) assign karta hai.
4. **Authority Dashboard:** Police ya NGOs ke liye ek command center.
5. **Live Safety Map (Heatmap):** Real-time crime tracking taake authorities hotspots par surveillance badha sakein.

## 💻 Tech Stack
- **Frontend:** Next.js, React, TailwindCSS, Leaflet Maps (for Heatmap), Spline (for 3D models).
- **Backend:** Python (FastAPI), SQLAlchemy.
- **AI Models:** Groq API (Llama-3 for logic, Whisper for Voice-to-Text).
- **Integrations:** Meta WhatsApp Cloud API.

## 📈 Impact / Future Scope
- **Impact:** System mein trust build karna. Justice process ko fast, unbiased, aur accessible banana.
- **Future:** Multilingual support (Sindhi, Pashto, Balochi), direct police dispatch integration, court evidence packaging.
